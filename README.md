# DocGPT – AI Document Intelligence Platform

DocGPT is a RAG-based AI document intelligence platform that enables semantic document search, contextual question answering, and AI-powered summaries using modern LLM workflows.

## Features

* AI-powered document understanding
* Semantic search using vector embeddings
* Context-aware question answering
* Intelligent document summarization
* PDF/document upload support
* FastAPI backend APIs
* Vector storage with ChromaDB
* LangChain-powered retrieval pipeline
* Production-ready deployment on AWS
* CI/CD automation using GitHub Actions

## Tech Stack

### Backend

* FastAPI
* Python
* LangChain
* ChromaDB

### AI & RAG

* Embedding Models
* Retrieval-Augmented Generation (RAG)
* Semantic Search
* Contextual AI Querying

### Cloud & DevOps

* AWS
* GitHub Actions
* Docker

## Architecture

Document Upload → Text Extraction → Embeddings Generation → ChromaDB Vector Storage → LangChain Retrieval → LLM Response

## Key Functionalities

### Semantic Search

Users can search documents using natural language queries instead of exact keyword matching.

### AI Summarization

Generates contextual summaries and extracts important insights from uploaded documents.

### Contextual Q&A

Answers user queries based on uploaded document context using Retrieval-Augmented Generation (RAG).
